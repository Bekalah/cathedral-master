# Cathedral of Circuits Modular Foundation

## Structure
- `site/index.html`: Main entry point, loads nodes from `/data/nodes.json`.
- `site/data/nodes.json`: All "nodes" (cards, labs, lore) as modular JSON objects.
- `site/assets/img/`: Art and icons for each node and hero imagery.
- `site/labs/`: Each node's interactive lab (HTML mini-apps).
- `site/assets/js/nodes.js`: Loads nodes and builds the UI, paired with `assets/js/calm.js` for reduced motion.
- `site/assets/css/atelier.css`: Styles the site.

## How to Add Content
- Add a new node to `site/data/nodes.json`.
- (Optional) Add art to `site/assets/img/`.
- (Optional) Add a lab to `site/labs/`.
- All logic is modular - just update JSON, no code changes needed!

## Safety
- Commit often and push to GitHub to avoid data loss.
- Keep everything modular for easy recovery and expansion.
